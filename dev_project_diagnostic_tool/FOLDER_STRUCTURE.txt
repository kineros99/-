╔════════════════════════════════════════════════════════════════════════╗
║  🔧 universal-diagnostic.js                                             ║
╚════════════════════════════════════════════════════════════════════════╝

PURPOSE:    Main diagnostic engine
SIZE:       ~1200 lines
LANGUAGE:   JavaScript (Node.js)

THIS IS THE CORE ENGINE!

RUN IT:
  node universal-diagnostic.js

WHAT IT DOES:
  1. Scans your entire project
  2. Analyzes all code files
  3. Detects issues
  4. Generates reports
  5. Creates AI context

OPTIONS:
  --format=html         Generate HTML report only
  --format=json         Generate JSON report only
  --format=markdown     Generate Markdown report only
  --format=all          Generate all formats (default)
  --auto-fix            Automatically fix issues
  --verbose             Show detailed output
  --generate-ai-context Generate AI context (default: true)

REQUIREMENTS:
  • Node.js v14+
  • No additional dependencies!

BENEFIT:
  WITHOUT context: 10+ messages to explain your project
