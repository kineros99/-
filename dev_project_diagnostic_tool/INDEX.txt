🚀 QUICK COMMAND REFERENCE
═══════════════════════════════════════════════════════════════════════════

BASIC RUN:
  node universal-diagnostic.js

AUTO-FIX ISSUES:
  node universal-diagnostic.js --auto-fix

HTML REPORT ONLY:
  node universal-diagnostic.js --format=html

VERBOSE OUTPUT:
  node universal-diagnostic.js --verbose

VIEW HTML REPORT:
  macOS/Linux:  open output/diagnostic-report.html
  Windows:      start output\diagnostic-report.html

  Step 4: Fail build if score too low
  Step 5: Upload reports as artifacts

🆘 NEED HELP?
═══════════════════════════════════════════════════════════════════════════

Problem: "node: command not found"
→ Solution: Install Node.js from https://nodejs.org/

Problem: "Permission denied"
→ Solution: Run chmod +x universal-diagnostic.js

Problem: Tool isn't finding my project files
→ Solution: Make sure you placed this folder in your project root

Problem: Health score seems low
→ Solution: Review the issues list in the HTML report

Problem: Want to understand what each file does
→ Solution: Read FOLDER_STRUCTURE.txt

Other questions:
→ Check SETUP_AND_USAGE.txt (Troubleshooting & FAQ sections)
